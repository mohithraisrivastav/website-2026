AR Asset Generator

Builds a matte-black framed, borderless print at true real-world scale for every artwork. Outputs one ar-files.zip with a .glb (Android) and .usdz (iPhone) per print.

Ready. Make sure this page is opened from the site root so images/ resolves.
After it downloads:
1. Unzip ar-files.zip at the site root.
2. It creates an /ar folder with all files.
3. Commit & deploy. The Shop AR button picks them up automatically.
Re-run only when you add or change a print.
